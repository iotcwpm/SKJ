Detailed Description

template<typename X, typename Y>
class Spline< X, Y >

Cubic spline interpolation class from http://shiftedbits.org/2011/01/30/cubic-spline-interpolation/

Modified slightly to add particular interfaces. Implementation not changed. Templated on type of X, Y. X and Y must have operator +, -, *, /. Y must have defined a constructor that takes a scalar.
