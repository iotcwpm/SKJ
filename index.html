ioskj: a simulation model of the Indian Ocean skipjack tuna fishery

Symbols

The convention used for assigning symbols is to use Greek lower case letters (e.g. `alpha`) for model parameters, Roman capital letters (e.g. `N`) for arrays of model variables, and Roman lower case letters for array subscripts (e.g. `N_{r,a,s}`).

Dimensions
`r` Region subscript
`w` West region

Dimensions

There are several model dimensions that are used to partition both the fish population and fishing catches.
