Indian Ocean skipjack model : yield curve and MSY

This document summarises output from the yield task which is primarily intended for checking MSY calculations and the shape of the yield curve. The shape of the yield curve is dependent on the model's parameters (in this case read in from parameters/input/parameters.cila ) including selectivity parameters (shown below).
